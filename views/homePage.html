HOW DOES THIS SYSTEM WORK?

A User first registers using the Registration option. If the user doesnt have access to a computer or a phone, another person also can register for the user. On registration a Unique QR code will be given to each person along with a UID(Unique Identification Number). The QR code will be scanned in various public places. Since data is being logged in various places, it will aid in backtraking in case someone catches the virus. If you have lost or forgotten your QR Code/UID a new one can always be generated. The old one will become invalid.
